Steps to use this Device Pool...

  1. Login to the Portal with your LDAP credentials
  2. Select the device Configuration you would like to use
  3. Verify if such a device is available to run your test
  4. If available, Submit an APK if needed and select the tests to be run
  5. If not available, place a request for one such device and try again with different configuration
  6. Run the tests on target device and get the results
